# 📊 SuperStore Sales Analysis Dashboard

## 📝 Project Overview

This project is based on the **SuperStore Sales Dataset** using Microsoft Excel.
The dataset was cleaned, analyzed, and transformed into an interactive **Sales Analysis Dashboard** using **Pivot Tables, Pivot Charts, KPI Cards, and Slicers**.

The dashboard provides meaningful business insights related to:

* Sales Performance
* Profit Analysis
* Regional Sales
* Category-wise Trends
* Monthly Sales Growth

---

# 🛠️ Tools & Technologies Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* KPI Cards
* Data Cleaning Techniques

---

# 📌 Dashboard Components

## 🔹 KPI Cards

The dashboard includes important KPIs such as:

* 💰 Total Sales
* 📈 Total Profit
* 📦 Total Quantity
* 🏷️ Total Discount
* 🛒 Total Orders

---

## 📊 Charts Used

### 1️⃣ Region-wise Sales

* **Chart Type:** Pie Chart
* Displays sales contribution by region.

### 2️⃣ Profit by Category

* **Chart Type:** Column Chart
* Compares profit generated by each category.

### 3️⃣ Monthly Profits

* **Chart Type:** Bar Chart
* Shows monthly profit performance.

### 4️⃣ Sales by Category

* **Chart Type:** Pie Chart
* Analyzes category contribution to overall sales.

### 5️⃣ Monthly Sales Trend

* **Chart Type:** Line Chart
* Tracks sales growth over time.
